#クラス分け問題(再帰例題)#  

*※再帰を使ってください。*  


###---------問題文---------###  
生徒が*N*人います。  
それぞれの生徒が自分のテストの点数をひとつずつ持っています。  
その集合を、*{S_0, S_1, S_2, ... ,S_N-1}* とします。  

この生徒たちを、2つのクラスに分けて入れようと思います。  
2つのクラスの平均点の差が最小になるように分けたときの、その差を求めなさい。  


###----------入力----------###  
*N* : 生徒数  
*S_0 S_1 S_1 ... S_N-1* : 各々のテストの点数（スペース区切り）。  

###----------出力----------###  
*R* : クラスの平均点の差の最小値  



###-----------例-----------###  
*入力*  
2  
80 82  


*出力*  
2  

*なお、問題はsample*.txtに記述してあるので参考までに。*
